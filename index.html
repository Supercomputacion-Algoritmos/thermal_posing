Thermal Imaging Domain Adaptation for Multi-person Pose Estimation

Lupión, M., De Toro-Castro, A., González-Ruiz, V., Sanjuan, Juan F., M. Ortigosa, Pilar
Ulster University, University of Almería

Human Pose Estimation using the proposed solution.

Abstract

Thermal cameras are increasingly deployed in privacy-sensitive smart environments and enable perception under adverse illumination, yet state-of-the-art multi-person human pose estimators trained on visible imagery fail dramatically on thermal data. This work introduces an unsupervised domain adaptation framework that transfers a pre-trained visible-spectrum pose estimator to low-resolution thermal imagery without retraining the pose model. We learn a domain adapter that maps thermal frames into a self-discovered, pose-estimation-friendly representation, optimized end-to-end through a differentiable evaluator network. The evaluator is trained on automatically generated supervision that discriminates plausible from implausible poses by leveraging pose completeness and silhouette consistency, thereby avoiding manual annotation and paired visible–thermal capture. The domain adapter is trained with a composite objective that balances structural fidelity, evaluator-based pose correctness, and heatmap-level pose consistency on the subset of confident samples. Experiments on a multi-person thermal dataset show that OpenPose coverage rises from 10.25% correct poses to 86-90% after adaptation, while missing-pose cases drop to below 1%. Using OpenPose-derived pseudo-ground truth, the adapted pipeline achieves a mean averaged precision greater than 0.97 on held-out users, with low keypoint error (around 3 pixels). These results demonstrate that robust thermal pose estimation can be obtained from existing visible-spectrum models via lightweight unsupervised adaptation, enabling practical deployment in privacy-preserving sensing without costly data collection or retraining.

Correctness loss is set to zero

L1 loss is set to zero

The OpenPose-friendly image is not compared with the original thermal image, evolving wihtout being restricted to maintain the same structure.

Pose loss is set to zero

The correct annotated poses are not compared with the inferred poses in the synthetic image.
